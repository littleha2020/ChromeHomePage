# Onecho - 即刻检索
***一个简单的检索主页***

## 由来

个人比较Chrome自带的简洁主页，支持添加自定义链接和更换壁纸，但是它把搜索框和google搜索引擎绑定在一起，无翻墙工具时换国内搜索引擎就失效了。

这催使我写了这个主页。
